# 情シス養成講座「ゼロから一人前の情シスへ」

IT初心者が **一人前の情シス(情報システム担当者)** になるための総合学習教材です。
動画・HTML教材・クイズ・ハンズオンラボ・分岐型ケーススタディ・ドリルの多媒体構成で、
「飽きずに最後まで続く」学習体験を提供します。

- 対象: PCの基本操作はできるが、業務としてのITは未経験の方
- ゴール: 中小〜中堅企業の情シス業務を一人で回せるレベル(約200時間 / 10パート・71章)
- スコープ外: プログラミング等の開発専門領域(運用自動化のシェルスクリプトは含む)

## 🚀 学習の始め方

**ブラウザで `portal/index.html` を開くだけ**(サーバー不要・スマホ対応・ダークモード対応)。

- 学習ポータルから全章の教材・クイズ・ケーススタディ・ドリルへ移動できます
- 進捗はチェックボックスで記録され、レベルバッジが育ちます(localStorage保存)
- **スマホから使う場合**: mainへマージすると GitHub Actions が自動で GitHub Pages に公開します
  (`.github/workflows/pages.yml`)。公開URLをホーム画面に追加すればいつでも学習できます
