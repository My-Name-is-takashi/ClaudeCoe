# 情シス養成講座「ゼロから一人前の情シスへ」

IT初心者が **一人前の情シス(情報システム担当者)** になるための総合学習教材プロジェクトです。
動画・HTML教材・インタラクティブクイズ・ハンズオンラボ・分岐型ケーススタディなど、
複数の媒体を組み合わせて「飽きずに最後まで続く」学習体験を目指します。

- 対象: PCの基本操作はできるが、業務としてのITは未経験の方
- ゴール: 中小〜中堅企業の情シス業務を一人で回せるレベル(約200時間 / 10パート・70章)
- スコープ外: プログラミング等の開発専門領域(運用自動化のシェルスクリプトは含む)

## 動作サンプル(実装済み)

ブラウザで開くだけで動きます(サーバー不要・スマホ対応・ダークモード対応)。

| ファイル | 内容 |
|---|---|
| `portal/index.html` | 学習ポータル: 全70章のカリキュラム一覧+進捗管理(localStorage)+レベルバッジ |
| `portal/quiz.html` | クイズエンジン: 4択・複数選択・穴埋めに対応、即時採点+全問解説付き |
| `quizzes/part3-ch05-vi.js` | サンプルクイズデータ(viエディタ章・12問) |
| `lessons/part3/3-05-vi.html` | サンプルHTML教材「3-5 viエディタ完全入門」 |
| `videos/scripts/` | 動画台本サンプル2本(絵コンテ+ナレーション全文。声はかわいい系女性ボイス「いろは」) |

> スマホで使う場合は、GitHub Pages などで `portal/index.html` を公開するとホーム画面から
> いつでも学習できます(リポジトリ設定 → Pages → ブランチ公開だけでOK)。
