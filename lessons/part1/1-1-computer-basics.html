Part 1 コンピュータとOSの基礎

1-1 コンピュータの仕組み(CPU・メモリ・ストレージ)

🎯 この章で学ぶこと
  • CPU・メモリ・ストレージの役割を「机と本棚」の比喩で説明できる
  • PCカタログのスペック表を読んで、用途に合うPCを選定できる
📺 動画①「パソコンの中身、ぜんぶ見せます」②「スペック表の読み方」とあわせてどうぞ。

1. 3大部品は「頭脳・机・本棚」

部品比喩役割足りないと…
CPU頭脳計算・処理をするすべての動作が遅い(処理待ち)
メモリ(RAM)机の広さ作業中のデータを広げる場所。電源OFFで消えるアプリを複数開くと極端に遅くなる
ストレージ(SSD/HDD)本棚ファイルやOSを保存する場所。電源OFFでも残る「空き容量不足」で保存・更新ができない

「遅い」の相談は、まずこの3つのどれが詰まっているかを見る——これが情シスの基本動作です(タスクマネージャーの見方は1-3章で)。

2. スペック表の読み方

例: 社用PCカタログ
CPU: Core i5-1335U(10コア)     ← 世代とコア数を見る。iの後の数字はグレード(3<5<7)
メモリ: 16GB                    ← 事務なら8GB、標準は16GB。あとから増設できない機種も多い
ストレージ: SSD 256GB NVMe      ← いまはSSD一択。HDDより桁違いに速い
ディスプレイ: 14型 FHD(1920x1080)
インターフェース: USB-C(PD対応)x2, HDMI, USB-A x2  ← 会議室・充電の運用に直結
用途目安構成ポイント
事務(メール・Office)Core i3〜i5 / 8〜16GB / SSD 256GB軽さ・バッテリー・価格重視
開発・データ処理Core i7 / 32GB / SSD 512GB〜メモリを最優先で盛る
デザイン・動画Core i7+GPU / 32GB / SSD 1TBGPUと色精度の高い画面

3. SSDとHDD、そして周辺の規格

  • SSD: 高速・静音・衝撃に強い。ノートPCは実質SSD一択
  • HDD: 容量単価が安い。バックアップ・大容量保管用として現役
  • USB規格: 形(Type-A/Type-C)と速度(USB 3.x)は別の話。「Type-Cなのに遅い」は普通にある
  • 映像出力: HDMI / DisplayPort / USB-C(Alt Mode)。会議室トラブルの主戦場
「メモリとストレージの区別」は、社員さんへの説明でも毎週使います。“机”と“本棚”の例え、そのまま使ってOKですよ。

  2. CPU・メモリ・ストレージを上の表と見比べ、「事務用として合格か」を判定してみる
📌 まとめ
  • CPU=頭脳、メモリ=机、ストレージ=本棚。「遅い」はこの3つから疑う
  • スペック表は「用途に足りるか」で読む。標準構成表を持つと選定が速い
