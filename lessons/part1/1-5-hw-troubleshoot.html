Part 1 コンピュータとOSの基礎

1-5 ハードウェアトラブルシュート

🎯 この章で学ぶこと
  • 切り分けの3原則(再現性・直前の変更・最小構成)を使えるようになる
  • 3大トラブル「起動しない・遅い・映らない」の切り分けフローを身につける

1. 切り分けの3原則

原則問いかけ例
① 再現性「毎回?ときどき?特定の操作で?」毎回→設定/故障、ときどき→熱/接触/負荷を疑う
② 直前の変更「昨日までは動いていた?何か変えた?」更新・ソフト追加・席替え(ケーブル)後の不調は変更が原因の筆頭
③ 最小構成「余計なものを全部外したらどうなる?」周辺機器を全部抜く、別ケーブル・別モニターで試す

あわせて重要なのがヒアリング力。「壊れた」「変です」を、事実(いつから・何をしたら・どんな表示)に翻訳するのが情シスの腕です。

2. 3大トラブルの切り分けフロー

A.「PCが起動しない」

電源ランプは点く?
├─ 点かない → 電源ケーブル/ACアダプタ/コンセント/バッテリー(=電源系)
├─ 点くが画面真っ暗 → メーカーロゴは出る?
│   ├─ 出ない → ハードウェア(メモリ/マザーボード)。ビープ音を確認しメーカー保守へ
│   └─ 出るのにOSが立ち上がらない → OS/ディスク系。回復環境・セーフモード・ディスク診断
└─ 起動途中で再起動ループ → 直前の更新を疑う(更新のアンインストール)

B.「PCが遅い」

タスクマネージャーで CPU / メモリ / ディスク のどれが100%?
├─ CPU → 犯人プロセスを特定(ウイルススキャン中?暴走?)
├─ メモリ → 常駐/ブラウザのタブ地獄 → 再起動と増設判断
├─ ディスク → HDDなら寿命/SSD化を検討。空き容量10%未満も要対処
└─ どれも平常 → ネットワークが遅い可能性(Part 2-8へ)/熱でクロック低下

C.「外部モニターが映らない」

① ケーブル・入力切替(モニター側のINPUT)を確認 ←最多原因
② Win+P で出力モード(複製/拡張)を確認
③ 別ケーブル・別ポート・別モニターで切り分け(どこを替えたら直る?)
④ それでもダメならグラフィックドライバー更新

3. 修理か交換かの判断

  • 保証期間内 → 迷わずメーカー保守(勝手に分解すると保証が飛ぶ)
  • 購入4〜5年超 → 修理より交換が合理的(標準更新サイクルを決めておく)
  • ディスク故障の疑い → 作業より先にデータ保全(バックアップ確認)
「まずケーブル」「まず再起動」は雑に聞こえて、統計的には最強の一手です。ただし再起動の前に「保存していない作業はないか」を必ず聞くこと!

📌 まとめ
  • 切り分けは「再現性・直前の変更・最小構成」の3原則
