1. スイッチの仕事 〜MACアドレスで交通整理〜

  • スイッチは「どのポートの先にどのMACアドレスがいるか」を学習し、宛先のポートだけにフレームを送る
  • L2スイッチ: MACアドレス(L2)だけで転送。同一ネットワーク内の交通整理役
  • L3スイッチ: ルーター機能も持つスイッチ。VLAN同士の橋渡し(VLAN間ルーティング)は普通これが担う
  • PoE: LANケーブルで電力も送る機能。無線APや監視カメラの設置で必須知識

2. VLAN 〜1台のスイッチの中に「仮想の仕切り」〜

VLANは、物理的には1つのネットワークを論理的に複数へ分割する技術です。目的は主に3つ: ①セキュリティ(部門分離・ゲスト分離) ②ブロードキャストの抑制 ③管理のしやすさ。

VLAN10 社員PC
192.168.10.0/24 VLAN20 サーバー
192.168.20.0/24 VLAN90 ゲストWi-Fi
192.168.90.0/24 → 社内へは通さない
ミライ商事のVLAN設計。ゲストはインターネットにしか出られない。
用語意味
アクセスポート1つのVLANだけが流れるポート。PC・プリンターをつなぐ
トランクポート(タグVLAN)複数のVLANをタグ付きで相乗りさせるポート。スイッチ間・AP向けの幹線
ネイティブVLANトランク上でタグなしのフレームが属するVLAN
# スイッチ設定の読み方(よくある表記の例)
interface GigabitEthernet0/1
 switchport mode access
 switchport access vlan 10        # ← このポートはVLAN10専用(社員PC用)
interface GigabitEthernet0/24
 switchport mode trunk
 switchport trunk allowed vlan 10,20,90   # ← 幹線。3つのVLANが相乗り
⚠️ 現場あるある: 「差し替えたらつながらない」の一定数はアクセスポートのVLAN違い。空きポートに挿す前に「そのポートは何VLANか」を確認する癖をつけましょう。

3. 無線LANの基礎

項目要点
規格Wi-Fi 5(11ac)→ 6/6E(11ax)→ 7(11be)。周波数は2.4GHz(遠くまで届くが混雑)と5GHz/6GHz(速いが障害物に弱い)
暗号化WPA2以上が必須、新規はWPA3推奨。WEPは即時撲滅対象
認証家庭=事前共有キー(PSK)。企業=802.1X(WPA2/3-Enterprise)で個人ごとに認証(退職者をすぐ無効化できる)
SSID設計社員用(社内VLANへ)+ゲスト用(分離VLANへ)の2本立てが基本形
設置設計APの位置=電波調査(サイトサーベイ)。チャネル重複と電子レンジ(2.4GHz)干渉に注意
「Wi-Fiが遅い」相談の犯人は、だいたい ①2.4GHzの混雑 ②AP1台に人が集中 ③電波の死角、の3兄弟。周波数とAP配置を見るだけで当たりがつきますよ。
💪 やってみよう(10分)
  1. スマホのWi-Fi設定で、自宅のSSIDが2.4GHzと5GHzのどちらにつながっているか確認する
  2. 会社(または自宅)に必要なVLANを3つ設計し、それぞれのサブネットを割り当ててみる(2-2章の復習)
📌 まとめ
  • スイッチはMAC学習で交通整理。VLAN間はL3スイッチが橋渡し
  • VLANは「仮想の仕切り」。アクセス/トランクの区別を読めるように
  • 企業Wi-FiはWPA2/3-Enterprise+ゲスト分離が基本形
