# Part 2 ネットワーク基礎と実践(全9章 / 約25時間 / レベル1)

## パート概要

情シス業務の質問・障害の過半はネットワークに帰着する。
「つながらない」に立ち向かえるよう、**概念(動画)→ 計算(ドリル)→ 実機確認(ラボ)→
切り分け(ケーススタディ)** の4段構えで徹底的に鍛える最重要基礎パート。

**到達目標**: 社内ネットワーク構成図を読み書きでき、「つながらない」の切り分けが一人でできる。

## 各章詳細

### 2-1 ネットワークの全体像とOSI参照モデル

- **学習ゴール**: 自宅・オフィスのネットワーク構成を絵に描ける。OSI 7階層とTCP/IP 4階層を対応づけて説明できる
- **教材構成**
  - 動画①「メールが届くまでの大冒険」(12分): 1通のメールがPCから相手に届くまでを階層ごとにアニメで追体験
  - 動画②「オフィスのネットワークを describe する」(8分): ONU・ルーター・スイッチ・AP・FWの役割
  - クイズ: 10問
- **内容トピック**: LAN/WAN / OSI 7階層は「住所→道順→荷造り」の比喩で / カプセル化 / 家庭とオフィスの構成差

### 2-2 IPアドレスとサブネット

- **学習ゴール**
  1. IPv4アドレスとサブネットマスク(CIDR表記)を計算できる
  2. ネットワークアドレス・ブロードキャスト・利用可能ホスト数を求められる
  3. プライベート/グローバルアドレスを判別できる
- **教材構成**
  - 動画「IPアドレスは住所、サブネットは町内会」(15分): 2進数変換から /24 /26 の分割までを図解
  - **ドリル「サブネット計算道場」**: ランダム出題・スコア制。「/26のホスト数は?」「192.168.10.130/26 のネットワークアドレスは?」等を反射で解けるまで反復
  - クイズ: 12問(設計判断問題を含む: 「拠点に60台。最小のサブネットは?」)
- **内容トピック**: 2進数と10進数 / CIDR / プライベートアドレス3レンジ / IPv6は概要のみ(表記と存在理由)

### 2-3 TCP/UDPとポート番号

- **学習ゴール**: TCPとUDPの使い分けを説明でき、主要ポート番号(20問レベル)を即答できる
- **教材構成**
  - HTML「TCP/UDP図解」: 3ウェイハンドシェイクの紙芝居図解、ステートの概念
  - **ドリル「ポート番号かるた」**: 80/443/22/25/53/110/143/3389/445… をフラッシュカード形式で暗記
  - クイズ: 10問
- **内容トピック**: コネクションの概念 / ウェルノウンポート / 「ポートが開いている」の意味 / netstat・ssコマンド入門

### 2-4 DNSの仕組み

- **学習ゴール**: 名前解決の流れ(スタブ→キャッシュ→ルート→権威)を説明でき、`dig`/`nslookup` で調査できる
- **教材構成**
  - 動画「DNSはインターネットの電話帳」(13分): 再帰問い合わせをキャラクター劇で表現
  - ラボ「digで名前解決を追いかける」(60分): `nslookup`→`dig +trace`→レコード種別(A/AAAA/CNAME/MX/TXT)確認→hostsファイル書き換え実験
  - クイズ: 10問
- **内容トピック**: レコード種別 / TTLとキャッシュ / 社内DNSと外部DNS / hostsファイル / 「DNSを疑うべき症状」パターン集

### 2-5 DHCP・NAT・ルーティング

- **学習ゴール**: IP自動配布と経路の仕組みを理解し、`ipconfig`/`ip` コマンドで自端末の状態を読める
- **教材構成**
  - HTML「DHCPの4ステップ(DORA)」「NAT/NAPTの変換表」図解
  - ラボ「自分の端末の通信経路を調べる」(45分): IP/GW/DNS確認 → ARPテーブル → `tracert` で経路確認 → デフォルトゲートウェイの意味を体感
  - クイズ: 10問
- **内容トピック**: DORA / リース期間と枯渇障害 / 固定IPと予約 / NAT/NAPT/ポートフォワード / デフォルトゲートウェイ / スタティックルートの読み方

### 2-6 スイッチ・VLAN・無線LAN

- **学習ゴール**: L2/L3スイッチの違い、VLANによるネットワーク分割の目的、無線LANの規格とセキュリティを説明できる
- **教材構成**
  - 動画「VLANでオフィスを仕切る」(12分): 部署別VLAN・ゲストVLAN分離をアニメで
  - HTML「無線LAN設計入門」: 規格(Wi-Fi 5/6/7)、チャネル設計、WPA2/WPA3、エンタープライズ認証(802.1X)概要
  - クイズ: 12問
- **内容トピック**: MACアドレスとARP / L2/L3スイッチ / タグVLAN(トランク)とアクセスポート / PoE / SSID設計(社内・ゲスト分離) / 電波干渉と設置

### 2-7 VPNとリモートアクセス

- **学習ゴール**: 拠点間VPNとリモートアクセスVPNの違い、スプリットトンネルの概念を説明できる
- **教材構成**
  - HTML「VPN図解」: IPsec / SSL-VPN、フルトンネルとスプリットトンネル比較図
  - クイズ: 8問(「在宅勤務者から“社内システムだけ繋がらない”」等の状況判断)
- **内容トピック**: 拠点間/リモートアクセス / 暗号化トンネルの概念 / ZTNAへの潮流(概要) / VPN起因のよくある問い合わせ

### 2-8 ネットワークトラブルシュート実践

- **学習ゴール**: 「つながらない」を層の下から順に切り分ける手順を身体で覚える
- **教材構成**
  - ラボ「切り分けコマンド演習」(90分): `ping`(自分→GW→8.8.8.8→ドメイン名)→ `ipconfig /all` → `nslookup` → `tracert` → `arp -a` を意図的に壊した環境(hosts改変・DNS変更等)で実施
  - ケーススタディ(分岐型・3シナリオ):
    1. 「全社インターネット不通」 2. 「特定の人だけ社内サーバーに繋がらない」 3. 「Wi-Fiが遅い」
  - HTML「切り分けフローチャート(1枚もの)」
- **内容トピック**: レイヤー順切り分けの型 / 名前解決とIP到達性の分離 / 「いつから・誰が・何を変えた」ヒアリング / エスカレーション判断

### 2-9 Part 2 修了テスト

- 40問 / 80%合格(サブネット計算の実技5問を含む)
