# Part 4 Windows ServerとActive Directory(全6章 / 約20時間 / レベル2)

## パート概要

日本企業の社内ITの背骨である Active Directory を運用できるようにするパート。
評価版 Windows Server を使った検証ドメイン構築で、入退社対応・権限管理という
情シスの日常業務を疑似体験する。

**到達目標**: 検証環境にADドメインを構築し、ユーザー・グループ・GPO・ファイルサーバー権限の
日常運用ができる。

**ラボ環境**: VirtualBox + Windows Server 2022 評価版(180日)+ Windows 11 クライアントVM。
メモリ8GB以下のPC向けに「画面写真でたどる紙上ラボ」代替版も用意する。

## 各章詳細

### 4-1 Windows Serverの役割と管理ツール

- **学習ゴール**: Windows Server が社内で担う役割(AD/DNS/DHCP/ファイル/印刷)と管理ツール群を説明できる
- **教材構成**
  - 動画「Windows Serverは会社の“総務部長”」(10分)
  - HTML「管理ツール図鑑」: サーバーマネージャー、MMC、RSAT、PowerShell(存在と用途のみ)
  - クイズ: 8問
- **内容トピック**: 役割と機能 / GUIとServer Core / ライセンス(CALの概念は費用感まで)/ Azure ADとの関係(概要)

### 4-2 Active Directory基礎とドメイン構築

- **学習ゴール**
  1. ドメイン・OU・ユーザー・グループの階層を設計図として描ける
  2. 検証ドメインを構築し、クライアントPCをドメイン参加させられる
- **教材構成**
  - 動画「Active Directoryは社員名簿+鍵束」(13分): 認証と認可、ドメインコントローラーの役割
  - ラボ「ミライ商事ドメイン構築」(150分): AD DS役割追加 → `mirai.local` 昇格 → OU設計(拠点×部署)→ ユーザー/グループ作成 → Win11をドメイン参加 → ドメインアカウントでログオン確認
  - クイズ: 10問
- **内容トピック**: ドメイン/ツリー/フォレスト / OU設計のセオリー / AGDLP(グループ設計原則)/ Kerberos認証(概要図のみ)/ AD連携DNSの重要性

### 4-3 グループポリシー(GPO)

- **学習ゴール**: GPOで端末設定を一括統制でき、適用結果を検証(gpresult)できる
- **教材構成**
  - ラボ「GPO演習」(120分): ①壁紙とスクリーンセーバーロック強制 → ②パスワードポリシー → ③USBメモリ書き込み禁止 → ④ドライブマッピング → `gpupdate /force` と `gpresult /r` で検証
  - クイズ: 10問(適用順序・継承のケース問題)
- **内容トピック**: GPOのリンクと継承(LSDOU)/ セキュリティフィルター / コンピューター構成とユーザー構成の違い / 「GPOが当たらない」切り分け

### 4-4 ファイルサーバーとアクセス権設計

- **学習ゴール**: 共有権限とNTFS権限の関係を理解し、部署共有フォルダーの権限を設計・実装できる
- **教材構成**
  - ラボ「共有フォルダー権限設計」(120分): 要件(全社共有/部署共有/役員限り)からグループ設計 → 共有作成 → NTFS ACL設定 → 別ユーザーで検証 → アクセス監査ログ有効化
  - HTML「権限設計パターン集」: やってはいけない例(個人指定ACL、Everyoneフルコン)も併記
  - クイズ: 10問
- **内容トピック**: 共有権限×NTFS権限の合成 / 継承と明示的許可・拒否 / AGDLP実践 / クォータ / ボリュームシャドウコピー(世代復元)

### 4-5 AD運用実務(入退社・棚卸し)

- **学習ゴール**: 入社・異動・退職のアカウントライフサイクルを、手順書に沿って事故なく実行できる
- **教材構成**
  - ケーススタディ(分岐型・3シナリオ):
    1. 「4月:新入社員20名の一括受け入れ」(準備リストの優先順位判断)
    2. 「急な退職者。アカウントをどうする?」(即時無効化・データ保全・メール転送の判断)
    3. 「監査で“退職者アカウントが残っている”と指摘された」(棚卸しプロセスの設計)
  - HTML「入退社チェックリスト テンプレート」(実務でそのまま使える形)
- **内容トピック**: アカウントライフサイクル / 無効化と削除の使い分け / 退職者データの保持ポリシー / 定期棚卸し / 特権アカウントの管理
