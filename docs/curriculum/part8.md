# Part 8 情シス実務スキル(全7章 / 約15時間 / レベル4)

## パート概要

技術以外で「一人前」を分けるスキル群。ベンダー管理・予算・稟議・規程・
プロジェクト管理・社内コミュニケーション。
一人情シス/少人数情シスが会社を動かすための「ビジネス側の武器」を渡す。

**到達目標**: 見積もりを比較して交渉でき、予算計画と稟議書を書け、
システム導入プロジェクトを完遂でき、非IT部門に伝わる説明ができる。

## 各章詳細

### 8-1 ベンダーマネジメント

- **学習ゴール**: 相見積もりの取り方・比較・交渉、契約書とSLAのチェックポイントを実践できる
- **教材構成**
  - HTML「ベンダー付き合いの教科書」: RFI/RFP、見積書の読み方(一式表記への対抗)、保守契約の中身
  - 演習「見積比較ワーク」: 同一案件の見積書3通(意図的に条件が揃っていない)を比較表に正規化し、確認質問リストを作る
- **内容トピック**: 見積条件の揃え方 / 値引きより条件交渉 / SLAとペナルティ / 保守範囲の境界(「それは別料金です」の予防)/ ベンダーロックインとの付き合い方

### 8-2 IT予算と稟議

- **学習ゴール**: 年間IT予算の構造を作れ、通る稟議書を書ける
- **教材構成**
  - HTML「IT予算の作り方」: ラン(維持)とチェンジ(投資)の分離、更新サイクル表からの予算化、TCOの考え方
  - 演習「稟議書ライティング」: 「無線APの全社更改(500万円)」の稟議書を書く。悪い例→添削観点→良い例の順で学び、最後に自分で書く(テンプレート付き)
- **内容トピック**: 減価償却とサブスクの費用計上の違い(概要)/ 費用対効果の示し方(リスク回避額の言語化)/ 却下された時のリカバリー(段階導入案)

### 8-3 社内規程とIT統制

- **学習ゴール**: 情報セキュリティ規程・PC利用規程などの体系を理解し、現実に運用できる規程を整備できる
- **教材構成**
  - HTML「社内IT規程の全体像」: 規程マップ(セキュリティポリシー/PC利用/SNS/持ち出し/BYOD)とサンプル条文
  - クイズ: 10問(「この行動は規程違反?」判定と、規程側に問題があるケースの見極め)
- **内容トピック**: 規程・基準・手順の3層 / 例外申請プロセスの設計(例外を潰すのではなく管理する)/ IT統制と監査(J-SOXの概要)/ 周知と教育のやり方

### 8-4 システム導入プロジェクト管理

- **学習ゴール**: 要件定義→選定→契約→導入→定着 の流れを、関係者を巻き込みながら完遂できる
- **教材構成**
  - ケーススタディ「勤怠システム導入プロジェクト」(分岐型・大型): キックオフから本稼働まで6フェーズ。要件ヒアリングの抜け、現場の抵抗、スケジュール遅延、データ移行トラブルなどのイベントに対処しながら進める
  - HTML「小さな会社のプロジェクト管理」: WBS・課題管理表・体制図の最小セット(テンプレート付き)
- **内容トピック**: 要件定義(現行業務の棚卸し)/ Fit&Gap / 並行稼働と切り替え判断 / 定着化(マニュアル・説明会)/ 「導入がゴールではない」

### 8-5 ドキュメンテーションと構成管理

- **学習ゴール**: 「自分が明日入院しても回る」状態を作るドキュメント体系を構築できる
- **教材構成**
  - 演習「引き継ぎドキュメント作成」: Part 3-11 で書いた構築手順書を、他人が読める運用手順書に改訂する(セルフレビューチェックリスト付き)
  - HTML「情シスドキュメント標準セット」: システム構成図・ネットワーク図・アカウント台帳・運用カレンダー・障害時連絡先の書き方と更新ルール
- **内容トピック**: 構成図の書き方(描画ツールと記法)/ パスワード管理(平文Excel禁止、パスワードマネージャー)/ ドキュメントの鮮度管理 / 属人化の解消

### 8-6 非IT部門に伝わる説明力

- **学習ゴール**: 専門用語を使わずに、経営層・現場それぞれに合わせてITの必要性・リスクを説明できる
- **教材構成**
  - 動画「その説明、伝わっていません」(10分): NG説明とOK説明の対比ドラマ(いろは先輩が添削)
  - ロールプレイ演習(選択式): 「役員に“サポート切れOSの更新予算”を3分で説明」「現場に“パスワード変更のお願い”を反発なく伝える」— 選択肢で説明を組み立て、相手の反応が変わる
- **内容トピック**: 相手の関心事に翻訳(経営=金とリスク、現場=手間)/ 比喩の使い方 / 「ダメです」と言わない代替案提示 / 社内広報文の書き方

### 8-7 Part 8 修了テスト

- 25問 / 80%合格(稟議書・説明文の良否判定問題を含む)
