# Part 0 オリエンテーション:情シスという仕事(全4章 / 約6時間 / レベル0)

## パート概要

学習を始める前の「地図」を渡すパート。情シスの仕事の全体像を掴み、
自分がこれから何を学ぶのか・なぜ学ぶのかを腹落ちさせる。
ここだけは知識ゼロで視聴できる設計とし、**全編を動画中心**にして学習のハードルを最小化する。

**到達目標**: 情シスの業務範囲を自分の言葉で説明でき、学習計画を立てられる。

## 各章詳細

### 0-1 情シスとは何か 〜会社のITを支える仕事〜

- **学習ゴール**
  1. 情シスの5大業務(インフラ/端末/SaaS/セキュリティ/ヘルプデスク)を挙げられる
  2. 「開発エンジニア」と「情シス」の違いを説明できる
  3. 舞台となる架空企業「ミライ商事」の環境概要を把握する
- **教材構成**
  - 動画①「情シスってどんな仕事?」(10分): 出社から退社までを追うアニメで5大業務を紹介
  - 動画②「ようこそミライ商事へ」(6分): 全編共通の舞台設定(従業員300名・3拠点・システム構成図)の紹介
  - クイズ: 5問(★基礎中心)
- **内容トピック**: 情シスの定義 / コーポレートITと開発の違い / ひとり情シス問題 / 本コースの主人公設定

### 0-2 情シスの1日・1年 〜リアルな業務カレンダー〜

- **学習ゴール**: 日次・月次・年次の定常業務と突発業務の違いを説明できる
- **教材構成**
  - 動画「情シスの1日密着(平常日/障害日)」(12分): 平常運転の日と障害発生日の2本立て構成
  - HTML「情シス年間カレンダー」: 入社シーズン・棚卸し・監査・予算策定など年間行事の図解
  - ミニ演習: 「4月1日に新入社員が20人入る。前月までに何を準備する?」を考えてから模範解答を見る
- **内容トピック**: 定常運用と突発対応 / 入退社対応 / 監査対応 / 繁忙期

### 0-3 このコースの歩き方

- **学習ゴール**: 学習ポータルの使い方を理解し、自分の学習スケジュールを作成する
- **教材構成**
  - HTML「学習ガイド」: 媒体の種類、進捗管理、レベルバッジ、推奨ペース(週次モデルプラン3種: 平日30分型/週末集中型/短期集中型)
  - 演習: 学習計画シートの記入(テンプレート付き)
- **内容トピック**: ポータル操作 / 学習サイクル(見る→読む→解く→触る)/ 挫折しない計画の立て方

### 0-4 情シスのキャリアと資格マップ

- **学習ゴール**: 本コースと公的資格(ITパスポート/基本情報)の対応関係を理解し、学習後のキャリア展望を持つ
- **教材構成**
  - HTML「資格・キャリアマップ」: 本コースの各パートと ITパスポート/基本情報技術者/CCNA/LPIC の出題範囲対応表
  - クイズ: ITパスポート型の例題10問(現在地の力試し。不正解でも問題ない旨を明示)
- **内容トピック**: 資格の位置づけ / 情シスのキャリアパス(マネージャー/セキュリティ/コーポレートエンジニア)

## パート修了条件

- 全動画視聴+クイズ2セットを70%以上
- 学習計画シートの作成完了
