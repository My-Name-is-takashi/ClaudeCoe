# Part 1 コンピュータとOSの基礎(全6章 / 約15時間 / レベル1)

## パート概要

情シスの土台となる「PCという箱」を理解するパート。
ハードウェアの仕組みからOSの役割、そして情シスの登竜門である
**キッティング(PCセットアップ)** までを実践する。

**到達目標**: PCの構成部品と役割を説明でき、Windows PCのキッティングと
ハードウェアトラブルの一次切り分けができる。

## 各章詳細

### 1-1 コンピュータの仕組み(CPU・メモリ・ストレージ)

- **学習ゴール**
  1. CPU/メモリ/ストレージの役割を「机と本棚」の比喩で説明できる
  2. PCのスペック表(カタログ)を読んで用途に合うか判断できる
  3. HDDとSSD、メモリ増設の判断基準を説明できる
- **教材構成**
  - 動画①「パソコンの中身、ぜんぶ見せます」(12分): 実機分解映像+3Dアニメで部品を紹介
  - 動画②「スペック表の読み方」(8分): 実在しそうな社用PCカタログ3枚を比較して選定する演習形式
  - クイズ: 10問(スペック表読解の実践問題を含む)
- **内容トピック**: CPU(コア/クロック)/ メモリとスワップ / SSD・HDD / GPU / 周辺機器とインターフェース(USB規格) / 「事務用・開発用・デザイン用」PC選定基準

### 1-2 OSの役割 〜WindowsとmacOSとLinux〜

- **学習ゴール**
  1. OSの4つの仕事(プロセス管理/メモリ管理/ファイルシステム/デバイス管理)を説明できる
  2. ファイルシステム(NTFS/APFS/ext4)とパスの概念を理解する
  3. 32bit/64bit、ライセンスとエディションの違いを説明できる
- **教材構成**
  - 動画「OSはPCの何をしているの?」(13分): タスクマネージャーを開きながらプロセス・メモリを実演
  - HTML「3大OS比較リファレンス」: Windows / macOS / Linux の思想・管理方法・社内での使い分け
  - クイズ: 10問
- **内容トピック**: カーネルとシェル / プロセスとスレッド / 仮想メモリ / ファイルパス(区切り文字・権限の違い) / Windowsエディション(Home/Pro/Enterprise)とボリュームライセンス

### 1-3 Windowsクライアント管理入門

- **学習ゴール**: Windows 11 Pro の管理機能(ユーザー、更新、デバイスマネージャー、イベントビューアー)を操作できる
- **教材構成**
  - ラボ「Windows管理ツール一周ツアー」(90分): ローカルユーザー作成 → 権限変更 → Windows Update制御 → イベントビューアーでログ確認 → タスクスケジューラ登録、までを自分のPC(またはVM)で実施
  - HTML「Windows管理チートシート」: `winver` `msinfo32` `eventvwr` 等の呼び出しコマンド集
  - クイズ: 8問
- **内容トピック**: ローカルアカウントとMicrosoftアカウント / UAC / Windows Update のチャネルと延期 / イベントログの見方(システム/アプリケーション/セキュリティ) / BitLocker概要

### 1-4 PCキッティングとMDM入門

- **学習ゴール**
  1. キッティングの標準工程(開梱→初期設定→ソフト導入→資産登録→引き渡し)を実施できる
  2. MDM(Intune等)による自動化の概念(Autopilot/ゼロタッチ)を説明できる
- **教材構成**
  - 動画「キッティング完全実演」(15分): 新品PCを想定した開梱から引き渡しまでのタイムラプス+ポイント解説
  - ラボ「キッティング手順書を作ろう」(60分): ミライ商事の標準PC設定手順書を、提供テンプレートを埋める形で作成
  - HTML「MDM入門」: Intune / Jamf の位置づけ、Autopilot の流れ図
- **内容トピック**: キッティングチェックリスト / マスターイメージとプロビジョニング / 資産管理台帳への登録 / ラベリング / MDMでできること(構成プロファイル、リモートワイプ)

### 1-5 ハードウェアトラブルシュート

- **学習ゴール**: 「起動しない」「遅い」「画面が映らない」の3大トラブルを系統立てて切り分けられる
- **教材構成**
  - ケーススタディ「ミライ商事 サポート受付日誌」(分岐型・3シナリオ):
    1. 「PCが起動しない」(電源→POST→OS の切り分けツリーを体験)
    2. 「PCが遅い」(リソース確認→常駐→ディスク→熱)
    3. 「外部モニターが映らない」(ケーブル→設定→ドライバー)
  - HTML「切り分けフローチャート集」(印刷して手元に置ける1枚もの×3)
- **内容トピック**: 切り分けの原則(再現性・直前の変更・最小構成) / ユーザーへのヒアリング術 / 修理と交換の判断・メーカー保守の使い方
